Orders
Attribute Type Description
order_id INT primary key
customer_id INT -
shopping_cart_id INT reference to the shopping cart of this transaction
date_of_purchase DATETIME -
total DOUBLE the total cost of the order in $USD
num_items INT total number of items in bought

Customer
Attribute Type Description
customer_id INT primary key
first_name VARCHAR(45) -
last_name VARCHAR(45) -
street VARCHAR(45) customer's shipping street
city VARCHAR(45) shipping city
postal_code INT shipping postal code
country VARCHAR(45) country of residence
phone_number INT -
email VARCHAR(45) -

Credit Card
Attribute Type Description
customer_id INT foreign key for its owner, Customer: customer_id
card_number INT credit card number
card_type VARCHAR(45) Discover, Bank of America, etc.
expiration DATETIME expiration date: month and year

Crayon
Attribute Type Description
crayon_id INT primary key
color VARCHAR(45) basic color tag i.e. 'yellow'
name VARCHAR(45) descriptive name i.e. 'wrong kind of snow yellow' for yellow
price FLOAT cost of one crayon
inventory INT the stock count

Shopping_Cart
Attribute Type Description
shopping_cart_id INT foreign key to Orders: shopping_cart_id
customer_id INT foreign key to Customer: customer_id
crayon_id INT foreign key to Crayon: crayon_id
